# QTradingBot

This project combines all the `QTradingBot` code into one repo, with the `backend` and `frontend` as submodules

This project [https://github.com/SkyNotion/qbot](https://github.com/SkyNotion/qbot) 

Frontend: [https://github.com/joshua1code/Q-bot-FD](https://github.com/joshua1code/Q-bot-FD)

Backend: [https://github.com/DaNutmeg/qbot-server](https://github.com/DaNutmeg/qbot-server)

It also contains the `Swagger UI` API documentation at `spec/api.yaml`

## Clone

To clone this project with all it's submodules

```bash
git clone --recurse-submodules https://github.com/SkyNotion/qbot.git
```

## Running

### For Frontend

Read `frontend/README.md`

### For Backend

Read `backend/README.md`

### For API specification

Set your webserver root directory to `THIS_REPO_PATH/spec/`
